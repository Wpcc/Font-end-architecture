# 注释

## 普通注释

```html
<!-- some html -->
```



## 条件注释

IE5~IE9这5个版本的IE浏览器还另外支持一种特殊的**if条件注释**

```html
<!--[if IE]> html语句 <![endif]-->
```

- IE视角

```text
<!--[if IE]>  	 条件注释前缀
html语句 			代码
<![endif]-->	 条件注释后缀
```

- 其他浏览器

```tex
<!--	条件注释前缀
[if IE]>html语句<![endif]	代码
-->	条件注释后缀
```
