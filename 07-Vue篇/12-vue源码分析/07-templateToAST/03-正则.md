# 正则解析

- attribute

```javascript
const attribute = /^\s*([^\s"'<>\/=]+)(?:\s*(=)\s*(?:"([^"]*)"+|'([^']*)'+|([^\s"'=<>`]+)))?/
```

```tex
([^\s"'<>\/=]+)   捕获key
(=) 捕获等于
([^']*) 捕获双引号值
([^']*) 捕获单引号值
([^\s"'=<>`]+) 捕获直接赋值
```

- dynamicArgAttribute

```javascript
const dynamicArgAttribute = /^\s*((?:v-[\w-]+:|@|:|#)\[[^=]+?\][^\s"'<>\/=]*)(?:\s*(=)\s*(?:"([^"]*)"+|'([^']*)'+|([^\s"'=<>`]+)))?/
```

```tex
((?:v-[\w-]+:|@|:|#)\[[^=]+?\][^\s"'<>\/=]*) 捕获key 如v-if:[hello] 中括号使用的非贪婪匹配
(=) 捕获等于
([^']*) 捕获双引号值
([^']*) 捕获单引号值
([^\s"'=<>`]+) 捕获直接赋值
```
